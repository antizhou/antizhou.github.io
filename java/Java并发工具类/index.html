CyclicBarrier

Semaphore

用来控制同时访问特定资源的线程数量，它通过协调各个线程，以保证合理的使用公共资源。

semapthore依赖于AbstractQueuedSynchronizer实现的同步器。可以用来做流量控制。

Exchanger

用于线程间的数据交换。它提供一个同步点，两个线程可以交换彼此数据。
